# Email Spam Detection with Machine Learning

## Objective
Build a spam detection model to classify messages as Spam or Ham.

## Dataset
SMS Spam Collection Dataset
https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- NLP

## Process
1. Data Cleaning
2. TF-IDF Vectorization
3. Model Training
4. Prediction

## Algorithm Used
Multinomial Naive Bayes

## Results
Achieved high spam detection accuracy.

## Conclusion
Machine learning effectively filters unwanted messages.
