# Iris Flower Classification

## Objective
Build a machine learning model to classify Iris flowers into Setosa, Versicolor, and Virginica species.

## Dataset
Iris Flower Dataset
https://www.kaggle.com/datasets/saurabh00007/iriscsv

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Steps
1. Load Dataset
2. Data Cleaning
3. Exploratory Data Analysis
4. Train/Test Split
5. Model Training
6. Prediction
7. Evaluation

## Algorithm Used
Random Forest Classifier

## Results
Achieved high classification accuracy on the test dataset.

## Conclusion
Machine Learning can accurately classify Iris flowers using petal and sepal measurements.
