# Car Price Prediction Using Machine Learning

## Objective
Predict the selling price of used cars using machine learning.

## Dataset
Car Price Prediction Dataset
https://www.kaggle.com/datasets/vijayaadithyanvg/car-price-predictionused-cars

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Seaborn

## Features
- Car Age
- Fuel Type
- Seller Type
- Transmission
- Kilometers Driven

## Algorithm Used
Linear Regression

## Results
Predicted car prices with good accuracy.

## Conclusion
Car age and present market price significantly influence resale value.
