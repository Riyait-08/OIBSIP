# Unemployment Analysis with Python

## Objective
Analyze unemployment trends across India and understand the impact of Covid-19.

## Dataset
Unemployment in India Dataset
https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Data Cleaning
- State-wise Analysis
- Monthly Trends
- Covid Impact Analysis
- Correlation Analysis

## Visualizations
- Histogram
- Heatmap
- Line Charts
- Bar Charts

## Results
Observed significant unemployment spikes during Covid-19.

## Conclusion
Data visualization helps identify unemployment trends and regional differences.
