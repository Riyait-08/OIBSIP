# Sales Prediction Using Python

## Objective
Predict product sales based on advertising expenditures.

## Dataset
Advertising Dataset
https://www.kaggle.com/datasets/bumba5341/advertisingcsv
## Technologies Used
- Python
- Pandas
- Seaborn
- Scikit-Learn

## Features
- TV Advertising
- Radio Advertising
- Newspaper Advertising

## Algorithm Used
Linear Regression

## Results
TV advertising showed the highest influence on sales.

## Conclusion
Businesses can optimize advertising budgets using sales prediction models.
